# The cerebellar components of the human language network  

Custom code for generating the figures from Casto et al. (2025). First-level modeling relied heavily on the publicly availible [CONN toolbox](https://web.conn-toolbox.org/resources/conn-extensions/evlab) EvLab extension, and second-level analyses relied heavily on the [SPM_SS toolbox](https://www.evlab.mit.edu/resources-all/download-spm-toolbox-for-individual-subject-analyses). Data to generate the figures are available at https://osf.io/y5t46. Please contact Colton Casto (ccasto at mit dot edu) if you have questions or run into issues with the code.  

## Cite as:  

**Casto, C., Poliak, M. Tuckute, G., Small, H., Sherlock, P., Wolna, A., Lipkin, B., D'Mello, A., Fedorenko, E. (2025). The cerebellar components of the human language network.**
